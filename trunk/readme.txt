== Description ==

This plugin helps you configure your WordPress website to use Leadhacker.
After setting up an account at Leadhacker.ru, you simply enter your Leadhacker project
code in the plugin's configuration page and you're ready to start improving your website
using Leadhacker.

Leadhacker is a dramatically easier way for you to improve your website through instant call back function. 
Convert your website visitors into customers and earn more revenue today!

You'll need an [Leadhacker.ru](http://www.leadhacker.ru) account to use it.

== Installation ==

Sign up at [Leadhacker.ru](http://www.leadhacker.ru).

1. Upload the Leadhacker WordPress plugin to your blog
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Leadhacker project code in the plugin's settings page. 

You're ready to start using Leadhacker!
